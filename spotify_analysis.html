Spotify Data Analysis

Top 10 Artists by Number of Tracks

This bar plot displays the top 10 artists based on the number of tracks they have in the Spotify dataset. Each bar represents a different artist, and its height corresponds to the number of tracks attributed to that artist.

Data Source: Spotify Music Dataset

Proportion of Songs Included in Spotify Playlists by Number of Artists

This horizontal bar plot illustrates the proportion of songs included in Spotify playlists based on the number of artists attributed to each song. Each bar provides insight into whether songs with a higher number of artists tend to be included in Spotify playlists more frequently or if songs with fewer artists have a higher likelihood of playlist inclusion.

Data Source: Spotify Music Dataset
